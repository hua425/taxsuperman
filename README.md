# taxsuperman

這是一個使用 Vite + React 建立的靜態遊戲專案，已調整為可透過 GitHub Actions 部署到 GitHub Pages。

## 本機開發

需要 Node.js。

1. 安裝相依套件：
   `npm install`
2. 啟動開發伺服器：
   `npm run dev`

## 部署到 GitHub Pages

1. 將程式推送到 GitHub 的 main 分支。
2. 在 GitHub Repository Settings > Pages 中，選擇「GitHub Actions」。
3. Action 會自動建置並部署到：
   `https://hua425.github.io/taxsuperman/`
